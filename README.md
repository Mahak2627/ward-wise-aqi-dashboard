# Ward-wise AQI Dashboard

A real-time air quality monitoring dashboard that provides ward-wise and zone-wise AQI analysis using CPCB aggregated data.

## Features
- Real-time AQI fetching
- Ward & zone-wise rankings
- AQI prediction
- Government & public advisories
- Auto-refresh dashboard

## Tech Stack
- Python
- Streamlit
- Pandas
- WAQI API

## How to Run
pip install streamlit pandas requests streamlit-autorefresh  
streamlit run app.py
